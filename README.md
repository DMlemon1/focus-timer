# Focus Timer - 专注计时器

一个美观的跨平台专注软件，支持自定义时间和白噪音。

## 功能特点

- 自定义专注/休息时间和循环次数
- 白噪音背景音效（雨声、海浪、森林等）
- 跟随系统时间自动切换白天/夜间模式
- 手动切换主题模式
- 阶段完成弹窗提醒
- Telegram 风格界面

## 下载安装

### Mac M4 版本

从 [Releases](../../releases) 页面下载 DMG 文件，双击安装即可。

### Windows 版本

直接下载 `Focus Timer.exe` 运行。
